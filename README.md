# sas_lab
## 安装依赖

要安装本项目所需的依赖，请遵循以下步骤：

1. **确保你拥有Python环境**：笔者的版本是3.10.2。

2. **安装依赖**：
   使用以下命令安装`requirements.txt`文件中列出的所有依赖库：
   ```sh
   pip install -r requirements.txt
   ```

3. **验证安装**：
   安装完成后，你可以使用以下命令来验证依赖是否已正确安装：
   ```sh
   pip list
   ```

请注意，如果你使用的是Python 2，请将上述命令中的`pip`替换为`pip3`。
